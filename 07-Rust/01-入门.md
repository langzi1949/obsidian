# 2. 初始化一个rust项目

使用Rust官方的`Cargo`包管理器和构建工具

## 2.1 初始化

```shell
# 创建一个标准的Rust项目结构
cargo new hello_world_project  
```

## 2.2 包依赖管理

- 声明依赖：通过编辑`Cargo.toml`文件声明项目依赖的外部库（称为 Crates）
- 下载依赖：当你运行`cargo build`或者`cargo run`时，Cargo会自动从值[crates.io](https://crates.io/) (rust的中央仓库)下载所需的依赖

```shell
# 下载依赖
cargo build
cargo run
cargo add <库名>
```

## 2.3 项目构建

- `cargo build` 编译构建项目（自动下载依赖并构建）
- `cargo check`  快速检查代码是否有编译错误，但不生成可执行文件
- `cargo clean`   顾名思义，清理项目编译生成的文件

## 2.4 运行项目

- `cargo run` : 先构建项目，然后运行生成的可执行文件

## 2.5 测试与文档

- `cargo test`  ：运行项目中所有的单元测试和集成测试
- `cargo doc`： 生成项目的文档，通常为HTML格式

## 2.6 发布与安装

- `cargo publish`  将你的rust库发布到中央仓库
- `cargo install` 安装任何发布在中央仓库的可执行文件
