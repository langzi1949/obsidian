
#  1. 尚雯婕学习法
尚雯婕作为复旦的学霸，她总结了一套学习英语的方法

1. 下载20分钟左右的感兴趣的音频内容  <span class="r">必须要有原文稿</span>
2. 直接开始听写，直到所有的词语都写出来，不会的单词先标注
3. 开始背诵，把音频内容全部背诵完毕
4. 开口复述，必须要跟原音频一样的语速


反复训练上面的方法，差不多10-12篇，差不多就出山了，一篇差不多要一个月的时间能够完成。

# 2. 100LS

100LS是韩国的一个学渣的方法，用了10个月做到了英语突破

1.   找一部合适的剧集，比如动画片，对话简单，而且重复较多，不开字幕的看一遍，只是通篇了解剧情
2.  打开中文字幕，确认意思
3.  英文字幕，查单词，背单词
4.  看着字幕进行跟读，一定要跟上原版的速度
5.  彻底关闭所有字幕，然后跟读，直到跟上原版的速度，至此大成。

一定要选择简单对话的剧情
