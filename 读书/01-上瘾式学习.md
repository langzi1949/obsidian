
这本书主要是讲如何高效的学习的，有具体的理论基础，同时也有很多实操的案例和方法论给予我们去锻炼。


# 1.那些无效或者相对低效的学习方法

- 反复阅读，跟我们之前提倡的『书读百遍，其义自现』有点相似
- 低效的归纳，甚至将原结论原封不动的摘抄下来，自我感觉良好的样子
- 用『高亮』笔去划重点，给我们一个感觉就是==划过就等于掌握了==的错觉

很惭愧的是，上面的几种方式还确实是我经常使用的学习技巧，可能形成了路径依赖，我个人感觉上面的方法，虽说没有那么高效，但至少也不是作者说的那么危言耸听般的无效。

# 2. 书中认为的高效学习法

1. **主动回忆**
2. **分散学习**
3. **精细提问与自我解释**
4. **交叉学习**


## 2.1 主动回忆

> 真正学习的人往往察觉不到自己学习的效果

方式有很多种：
1. <span class='r'>自己在一个空白的纸上默写下之前记忆的内容，不参考任何的东西，纯粹的调用大脑中的IO，不需要任何的提示</span>
2. 做练习册或者试卷
3. 参加模拟考试
4. 给其他人讲一下你所学的内容，相当于做一个『学徒效应』，这个有点类似『费曼学习法』

## 2.2 分散学习

分散学习很好理解，就是不建议一次性集中记忆太多的东西或者内容，用一个数学的表达方式就是

$$5 <<< 2 + 2 + 1 (一次学习5，不如分散的学习2 + 2 + 1)$$

这个就有点类似之前艾宾浩斯遗忘曲线逻辑，最佳的实践就是
**第一次复习在1天后，第二次复习在7天后，第三次复习在16天后，第四次复习在35天后**


用上面的**主动学习** + **分散学习**，我认为是现阶段最强的记忆方式，多次主动学习将所有内容都记住才算一次学习，然后通过分散学习进行往复循环就行。


## 2.3 精细提问与自我解释

> 一言以蔽之： 通过在脑子内自问自答进行学习

针对一个知识点，可以不断的追问『为什么』？ 其实就是一种刨根问底的方式，然后在学习的过程中用**自己的话**向自己解释正在发生的事情以及逻辑步骤。

- 预习阶段，进行精细化问题，这就是我们学生时代老师说的，带着问题来学习，会更加高效
- 执行阶段，进行自我解释，就是对问题进行解答，不能照本宣科，而是有自己的理解
- 复盘阶段，需要针对这些问题和答案，我们去探索是否这些就是正确的

## 2.4 交叉学习

很好理解，多个知识点或者课程，进行混合学习，不要长时间的学习一种东西，要穿插着学习其他的知识，这样就可以提高效率。


后面两种的学习方式，其实不用刻意的追求，但是**主动回忆** + **分散学习** 绝对是我们要追求，要严格执行的。

# 3. 调整状态

- 明白自己学习的目标是啥
- 可以换不同的地点学习，可以在书房、在图书馆、公园等地点不同切换

中国有句古话，『谓学不暇者，虽暇亦不能学矣』，意思就是那些说自己没有时间学习的人，即使有时间他们也不会学习。

# 4. 善于记笔记

主流的方式有
- 康奈尔笔记法
- 卡片盒笔记法

## 4.1 康奈尔笔记法

这个是目前最均衡的全面型的笔记法
1. 布局
	1. **将纸张分为3个区域，左侧是『线索栏』，右侧是『笔记栏』，底部是『总结栏』**
2. 操作
	1. ==记录 Record== ，在右侧记录核心内容
	2. ==简化 Reduce==   课后立即在左侧写出关键词或者提问
	3. ==背诵  Recite==   遮住右侧，看看左侧的问题进行自我解释
	4. ==思考 Reflect==   将新知识和旧知识联系
	5. ==复习 Review==    定期看左侧的线索栏进行回想
