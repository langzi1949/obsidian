
# 1. 什么是智能合约？

**一句话总结：智能合约 = 部署在区块链上的自动执行的代码**

智能合约像一个机器人，当满足条件时自动执行，无需人工干预，无法篡改。

简单的对比

| 传统系统 <img style="width:250px"/> | 智能合约 <img style="width:250px"/> |
| ------------------------------- | ------------------------------- |
| 代码部署在服务器                        | 代码部署在区块链                        |
| 后端可随时改代码                        | 一旦部署不能修改                        |
| 数据库存储在企业                        | 数据存储在区块链                        |
| 代码由公司空值                         | 代码开源，公开透明                       |
| 费用由公司承担                         | 消耗Gas费用                         |

# 3. 什么是Gas？

在区块链，尤其是ETH EVM链里面： **Gas = 区块链上执行代码的『计算费用』**

在链上运行代码、修改数据、调用智能合约，都需要花钱，这个钱就是Gas
区块链是分布式的，每次执行代码，全球所有节点都必须
1. 同时执行这段逻辑
2. 达成共识
3. 将结果保存进区块
以上的都需要钱。。。。。

# 5. 如何发币

在区块链的世界，任何人都可以发行虚拟货币（Token）,门槛低到令人发指的地步，但是能发行 不等于 有价值

为什么可以发行虚拟币？ 因为大多数链允许部署自己的智能合约。
代币本质就是
> 一份智能合约，记录每个地址的余额。

它不需要
- 不需要公司
- 不需要审查审批
- 不需要审查
- 不需要任何权限

## 5.1 发行虚拟币需要什么？

1. 一个钱包（MetaMask）
	1. 保存你的私钥，支付部署费用Gas
2. 一点点手续费
	1. 在ETH部署一个ERC20，大概需要20-80个 USD Gas
3. 一份基于ERC20合约
	1. 自己写

## 5.2 怎么发布一个虚拟币呢？

目前最主流的方式是 用`openzeppelin` + `Remix`

1. 安装一个钱包
	1. 最常用：MetaMask
	2. 这个是发布合约用到的钱包
2. 准备一点ETH  部署费用
3. 写智能合约
	1. 设置名称、符号、初识发行量
4. 上传代码到Remix网站
5. 选择链并部署合约
	1. 确认MetaMask交易 -> 链上生成一个新的Token合约
6. 合约地址生成，代币已经发布成功
	1. 每个人可以查询代币的余额、转账、添加到MetaMask钱包

## 5.3  什么是OpenZeppelin、Remix、Hardhat

 OpenZeppelin是web3和智能合约开发领域中最权威、最重要且使用最广的开源组织和代码库之一
简单来说，OpenZeppelin就是智能合约世界中的『安全基石』和『标准模板库』
开发者不要写那些基础代码，直接继承或者导入OpenZeppelin提供的安全代码模块

Remix 是在线IDE，用于智能合约的开发

Hardhat 是用于以太坊和EVM及其兼容区块链的开发环境和框架，提供了一套专业的、基于命令行的、功能强大的本地开发工具套件。

# 6. 当前区块链的弊端和不足的地方

区块链的难点从来不是场景，而是技术问题，有多个根本性的技术问题没有得到解决

1. **性能问题，TPS太低，无法大规模应用**
因为区块链必须全网节点同步数据和验证交易，这个就导致TPS很低，延迟很高，吞吐量极低
Visa TPS在2W+，微信支付宝在1W+，以太坊只有15-30，比特币更低，只有7

2. **拓展性差**
无法同时兼顾  去中心化 + 安全 + 高性能

3. **成本高**

链上每次计算都需要费用，普通用户根本用不起

4. **链下的数据不安全，导致无法构建大型应用**
5. **安全风险极高**
智能合约写错代码，由于无法回滚，可能导致巨额亏损
